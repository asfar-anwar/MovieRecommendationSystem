# Movie Recommendation System

This is a simple Python project that recommends movies based on the genre selected by the user.

Movies are sorted by rating so that the highest rated movies are shown first.

---

## Technologies Used
- Python
- Pandas

---

## How to Run the Project

Step 1: Install required library  
pip install -r requirements.txt  

Step 2: Run the program  
python recommend.py  

---

## Features
- Genre-based movie recommendations
- Movies sorted by rating
- Easy to understand for beginners
- Uses CSV dataset

---
